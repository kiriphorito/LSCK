Prototyping


In this page we will talk about our client and the task that they have assigned us.

Initial Requirements

The clients for this project are the Visual Studio team from Microsoft and Dean Mohamedally. The Visual Studio Team at Microsoft produces a variety of software and services for developers around the world to use such as Visual Studio Team Services (VSTS), Visual Studio Code and Visual Studio IDE. We are developing an extension for Visual Studio IDE.

Gathering Requirements

Imagine there is a new start-up that wishes to showcase their new project with a few snippets of code and paraphrasing other sections. They could have to take time out of their schedule to first trawl through their code base to find the snippets of code, add comments or a paraphrase and with this develop a website that include this along with images and videos.

This time could have been spent further developing the project and an extension could have generated the website. All you had to do was link the files and add tags to the code that you wanted to include as a snippet. This is what we want to investigate and achieve.
