Introduction

The aim of the lightweight practitioner kit is to investigate and develop an extention for Visual Studio which allow developers whether it be individuals or small organisations, to showcase their achievements and features of a project in a portfolio that can be generated in a click of a button. The portfolio will be in a HTML format and hosted to allow you to present the portfolio to potential employers or to display to the world. Making it easier for developers to concentrate on their projects rather that spending precious time on a website.
