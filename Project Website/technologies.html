Technologies Used


In this page we will discuss about the different technologies that have been used inside LSCK

LSCK Extension

The extension was developed for and in Microsoft Visual Studio 2015. It was also programmed in the language C# (instead of Visual Basics). Along with this, there were a number of NuGet packages used in this project which are JSON.NET and SSH.NET.

Link below for the technologies:

  • Microsoft Visual Studio - https://www.visualstudio.com/
  • C# - https://msdn.microsoft.com/en-gb/library/67ef8sbd.aspx
  • Json.NET - http://www.newtonsoft.com/json
  • SSH.NET - https://github.com/sshnet/SSH.NET

Website Portfolio

For the generated website, the page was using HTML. CSS was used for design and user customisations. Bootstrap was used to make the website look and feel more professional and Ace editor allows you to show code in an code editor that is embedded into the website giving a more professional feel.

Link below for the technologies:

  • HTML - https://html.spec.whatwg.org/multipage/
  • CSS - https://www.w3.org/Style/CSS/specs.en.html
  • Bootstrap - http://getbootstrap.com/
  • Ace Editor - https://ace.c9.io/

Uploading and Hosting

For hosting we concentrated on using Linux(Unix) as it holds the largest market in operating systems used for hosting. Then we decided to use Apache for the actual hosting software.

Link below for the technologies:

  • Linux - https://en.wikipedia.org/wiki/Linux
  • Distro of Linux - Ubuntu - https://www.ubuntu.com/
  • Apache - https://httpd.apache.org/
